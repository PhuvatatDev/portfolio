# PhuvatatDev — Portfolio

Personal developer portfolio. One-page scroll-animated website showcasing my work, skills, and featured project [MindTarot](https://mindtarotapp.com).

**Live:** [phuvatatdev.github.io/portfolio](https://phuvatatdev.github.io/portfolio)

## Stack

- **Astro 5** — Static site generation
- **TypeScript** — Strict typing
- **Tailwind CSS** — Utility-first styling
- **GSAP + ScrollTrigger** — Scroll-driven animations

## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Animated marquee, tech stack logos, call-to-action |
| **My Work** | 4-step process (Idea → Discussion → Build → Live) with tech marquee |
| **Featured Project** | MindTarot — phone frame with video demos + interactive tech cards |
| **Repositories** | Project table (live apps, practice repos, client work) |
| **About** | Photo carousel + story (self-taught, from no-code to production) |
| **Contact** | Email click-to-copy, GitHub link, tech marquee |

## Featured Project — MindTarot

An introspective tarot app with AI-powered interpretations. Built from authentication to payment system to store deployment.

**Stack:** Flutter · Firebase · Gemini AI · RevenueCat · Clean Architecture

- [Google Play](https://play.google.com/store/apps/details?id=com.pvtdev.mindtarot)
- [App Store](https://apps.apple.com/app/mindtarot/id6757782630)
- [Website](https://mindtarotapp.com)

## Run Locally

```bash
npm install
npm run dev       # localhost:4321
npm run build     # Static build → dist/
```

Requires Node 20+.
